################### INSTALAÇÂO ########################

Abra o arquivo "connect.php" (localizado no diretório principal) e defina as configurações do banco de dados (host, usuário e senha).
Caso o projeto seja aberto localmente, o mesmo já encontra-se configurado, sendo necessário apenas fazer a importação do banco de dados que encontra-se no diretório "BD" com nome "contato.sql".

**Obs.: Se estiver definido uma senha para o root locamente, informe a mesma no arquivo "connect.php".
